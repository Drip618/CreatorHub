# Creator Hub 终极使用手册

欢迎使用 **Creator Hub** —— 专为顶级剪辑师与自媒体创作者打造的 macOS 全能生产力工具。本手册将带您深度解析这款“工作流加速器”的所有核心功能。

---

## 🚀 软件概览
Creator Hub 是一款集成了图像处理、视频增强、文档转换与自动化脚本的系统级工具。它通过无缝的后台常驻与全局快捷键，将原本繁琐的多步骤操作简化为“一键式”体验。

---

## 🛠️ 核心功能模块

### 1. 🖥️ 屏幕与图像专家
*   **屏幕截图**：快速捕捉全屏图像并自动保存至指定目录。
*   **文字识别 (OCR)**：精准提取屏幕任何区域的文字，自动存入剪贴板。
*   **隐私净化**：批量清除图片中的 EXIF 信息（地理位置、拍摄设备等），保护创作隐私。
*   **万能图像处理**：批量转换图片格式、调整尺寸，适配各种社交媒体平台。

### 2. 🎬 多媒体创作引擎
*   **终极视频工坊**：集成 FFmpeg 核心，支持极速视频压缩、抽帧、无损去水印。
*   **全网媒体解析**：只需复制链接，即可一键解析并下载主流平台的素材视频。
*   **音频标准化**：自动平衡音频音量至 -14 LUFS 行业标准，解决音量忽大忽小的问题。
*   **ProRes 级转码**：专为后期优化，批量转换代理文件。

### 3. 📄 文档与脚本流
*   **语音听写**：将灵感实时转为文字，支持长篇口述记录。
*   **多语言互译**：剪贴板内容的即时翻译，适配国际化创作。
*   **宇宙文档转换**：基于 Pandoc 引擎，支持 Word、Markdown 等格式的完美互转。
*   **XML 版本降级**：解决 FCPX 或 PR 项目工程版本不兼容导致的打不开问题。

### 4. 📊 智能生产力
*   **屏幕取色**：精准获取 UI 颜色码，设计与剪辑调色的必备工具。
*   **库缓存清理**：一键清理剪辑软件冗余的渲染文件，释放数百 GB 磁盘空间。
*   **JSON 专家工具**：将乱序文本或 Key-Value 数据瞬时转为标准化的 JSON 结构。

## ⌨️ 全局快捷键配置
软件支持 3 个自定义快捷键插槽，默认配置为：
*   **⌥1 (Option + 1)**：默认分配 [屏幕截图]
*   **⌥2 (Option + 2)**：默认分配 [文字识别]
*   **⌥3 (Option + 3)**：默认分配 [多语言翻译]

**设置方法：**
1.  进入“偏好设置”选项卡。
2.  在对应的“功能快捷键”下拉菜单中选择您最常用的功能。
3.  更改后即时生效，无需重启。

---

## 📂 文件管理
*   **默认保存位置**：您可以在设置中更改文件下载与导出的路径。
*   **自动分类**：下载的素材会自动归类至 `Test_Downloads` 文件夹，确保桌面整洁。

---

## ❓ 常见问题
*   **权限申请**：首次使用 OCR 或录屏功能时，请确保已在系统“安全性与隐私”中授予相应权限。
*   **环境变量**：软件已内置 FFmpeg 与 Pandoc，无需额外安装环境，真正做到开箱即用。
