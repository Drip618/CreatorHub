# Creator Hub 使用说明 (测试版)

感谢参与 Creator Hub 的内部测试！为了确保您能获得最佳的使用体验，请在第一次运行时按照以下步骤操作：

## ⚠️ 首次运行必读 (权限设置)

Creator Hub 是一个系统级工具，需要以下权限才能发挥全部功能：

1.  **辅助功能 (Accessibility)**:
    *   **作用**: 实现“划词翻译”功能（自动模拟复制）。
    *   **设置方法**: 运行软件后，若弹出权限提示，请点击去设置；或打开【系统设置 -> 隐私与安全性 -> 辅助功能】，勾选 **Creator Hub**。
2.  **屏幕录制 (Screen Recording)**:
    *   **作用**: 用于 OCR 文字识别和截图。
    *   **设置方法**: 在【系统设置 -> 隐私与安全性 -> 屏幕录制】中开启。

---

## ⌨️ 快捷键指南

*   **Option + 1**: 屏幕截图 (自动打开预览编辑)
*   **Option + 2**: 屏幕文字识别 (OCR)，结果会弹出悬浮窗
*   **Option + 3**: **划词翻译** (在任何地方选中文字，按此键即刻翻译)

---

## 🖱️ 右键菜单集成

您可以选中一个或多个图片/视频文件，点击**鼠标右键 -> 快捷操作 (Quick Actions) -> 多媒体极速处理 (Creator Hub)**，即可快速进行视频压缩、合并 PDF 或清洗图片隐私。

---

## 📂 保存目录

软件默认会将处理后的文件保存在您的 **「下载 (Downloads)」** 文件夹中。您可以在顶部菜单栏点击图标 -> 设置 -> 自定义保存地址。
